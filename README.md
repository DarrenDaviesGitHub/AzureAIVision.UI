Front End UI developed with React, NextJS, TypeScript and Motion to consume the [https://github.com/DarrenDaviesGitHub/AzureAIVision](Azure AI Vision API).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
